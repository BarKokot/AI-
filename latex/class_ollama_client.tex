\doxysubsection{Detailed Description}
Obsługuje komunikację z API serwera Ollama przez REST. 

Klasa buduje zapytania JSON i wysyła je synchronicznie do endpointu /\+api/\+generate działającego serwera Ollama. Przeznaczona do użycia wewnątrz wątku roboczego (\doxylink{class_ollama_worker}{Ollama\+Worker}).
