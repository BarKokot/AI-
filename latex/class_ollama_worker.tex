\doxysubsection{Detailed Description}
Wykonuje zapytania do Ollama w osobnym wątku (QThread). 

Dzięki uruchomieniu w tle zapytania do modelu językowego nie blokują interfejsu graficznego. Po zakończeniu emituje sygnał \doxylink{class_ollama_worker_a17a40e181c6505f26a55ecf5d6043d79}{finished()}.
