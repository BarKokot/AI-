Detailed Description

Wyodrębnia kod Python z odpowiedzi LLM i wykonuje go.

Kod jest zapisywany do pliku tymczasowego i uruchamiany przez interpreter Python. Wykres jest zapisywany do PNG i wczytywany do zakładki Wykres.
